# 玩具店

玩具店官網，提供商品展示與後台管理功能。

## 快速開始

```bash
# 後端
cd backend
npm install
npm run dev   # http://localhost:3001

# 前端
cd frontend
npm install
npm run dev   # http://localhost:5173
```

## 檔案說明

```
toy-store/
├── README.md         ← 這份說明
├── TODO.md           ← 待辦事項
├── CLAUDE.md         ← AI 助理設定（不需要動）
├── .env.example      ← 環境變數範本
├── .gitignore
├── frontend/         ← React 前端
├── backend/          ← Node.js + Express 後端
└── docs/
    ├── PROJECT.md    ← 專案說明
    ├── API.md        ← API 規格
    ├── DATABASE.md   ← 資料表結構
    ├── UI_RULES.md   ← UI 設計規範
    ├── PAYMENT.md    ← 金流串接（未來使用）
    └── CHANGELOG.md  ← 版本紀錄
```
